# B2C Quick POS

Commercial Point of Sale module for Perfex CRM.

Version: 1.0.0-beta  
Author: Sherwin Armas

## Overview

B2C Quick POS provides fast cashier operations, split payments, reports, exports, and customer rewards in a single module.

## Core Features

- Cashiering terminal with barcode-ready product search
- Split payment checkout with validation
- Quick customer creation at checkout
- POS invoice creation with payment posting
- Cashier report and Z report views
- CSV export for reports
- PDF export for Z report
- Rewards system with earn/redeem logic
- Reward expiry settings and transaction history
- Permissions for cashiering and report access

## Requirements

- Perfex CRM 2.9.0+
- PHP and MySQL environment supported by your Perfex version
- Admin access for installation and permission setup

## Module Folder

This module should be placed at:

modules/pos

## Installation Summary

1. Upload the module folder to modules/pos.
2. Activate the module from Perfex Modules.
3. Configure permissions for staff roles.
4. Open POS menu and verify cashiering/reports/rewards pages.

Detailed steps are in INSTALLATION.md.
