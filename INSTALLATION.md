# Installation Guide

## 1. Pre-Install Checklist

- Confirm Perfex CRM version is 2.9.0 or higher.
- Confirm you have admin access.
- Backup database and files.

## 2. Upload Module

- Upload the pos folder to:
  modules/pos

## 3. Activate Module

- Log in as admin.
- Open Setup -> Modules.
- Activate B2C Quick POS.

## 4. Configure Permissions

- Go to staff role permissions.
- Enable needed POS capabilities:
  - POS Cashiering
  - POS Cashier Report (All Staff) or (Own)
  - POS Z Report (All Staff) or (Own)
  - POS Rewards Settings

## 5. Verify Basic Operation

- Open POS Quick Sell.
- Add products to cart.
- Complete a checkout.
- Verify invoice and payment records.
- Open report pages and test CSV/PDF exports.

## 6. Rewards Setup

- Open Rewards Settings.
- Enable rewards if needed.
- Configure point earn rates and redemption value.
- Configure point expiry policy.

## 7. Post-Install Recommendation

- Test with a non-admin cashier role.
- Review report totals for expected behavior.
- Keep module and Perfex updated.
