# Support

Author: Sherwin Armas
Version: 1.0.0-beta

## Support Scope

Support covers:

- Installation guidance
- Module configuration guidance
- Defect fixes in included features

Support does not cover:

- Custom feature development not listed in package scope
- Third-party customizations outside this module
- Server administration unrelated to module functionality

## Before Requesting Support

Please provide:

- Perfex CRM version
- PHP version
- Clear issue steps
- Screenshots or error messages
- Relevant server log excerpts

## Best Practice

Always test updates in staging before production rollout.
